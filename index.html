Why Should I Care About Testing?


  • Regression
  • Refactoring
  • Updates / Upgrades
  • Monitoring
  • CI/CD
  • Bug reproduction environment
  • 22/25 trending Github projects have tests! (https://github.com/trending)

Why xDD

  • Developer, who wrote the code, knows best what it should be doing
  • Better fit for Agile
  • Encourages: encapsulation, Deterministic system setup / teardown
  • Self documenting

Why BDD

  • "Value" driven
  • Focus on behavior, not code coverage
  • Common (domain specific) language for communicating
  • User stories are aligned with delivered features
